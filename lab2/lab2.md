# Лабораторна робота №2. Індексація та алгоритми Google

## Мета
Навчитись перевіряти стан індексації сайту через Google Search Console, зрозуміти статуси
Coverage Report, впровадити E-E-A-T-сигнали у власний проєкт та отримати базовий Lighthouse звіт як точку відліку для
подальшої оптимізації.

## 1. Скріншот URL Inspection з GSC (головна сторінка)

| Параметр | Значення |
| :--- | :--- |
| **Статус індексації** | URL-адреса є в Google (Сторінку можна проіндексувати) |
| **Дата останнього crawl** | 8 квітня 2026 р., 17:32:04 |
| **Метод виявлення URL** | Визначається після індексації (у Live Test: Не перевірено) |
| **Чи дозволено індексацію robots.txt** | Так (Дозволено) |
| **Чи є canonical** | Користувачем: Жодних; Google: Визначається після індексації |
| **Статус рендерингу** | Успішно (Google має доступ до цієї URL-адреси) |
