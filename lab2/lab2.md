## 2. Результати site:, cache:, info: операторів

Нижче наведено фактичні результати перевірки видимості домену в пошуковій системі Google за допомогою спеціальних операторів.

| Оператор | Результат (на скріншотах) | Що це означає |
| :--- | :--- | :--- |
| **`site:pop-collectors.pp.ua`** | Знайдено 1 результат (головна сторінка). | Сайт успішно проіндексований і вже є в базі Google. |
| **`cache:pop-collectors.pp.ua`** | Нічого не знайдено (напис "не знайдено жодного документа"). | Кешована копія сторінки в Google наразі відсутня. |
| **`info:pop-collectors.pp.ua`** | Звичайна пошукова видача. | Наш сайт знаходиться на 1-му місці в результатах пошуку. |

## 3. Заповнена таблиця статусів Coverage Report 

| Статус | Пояснення | Можлива причина |
| :--- | :--- | :--- |
| **Submitted and indexed** | Сторінку з sitemap успішно знайдено і додано в пошук. | Якісна картка товару (Funko Pop), яка додана в `sitemap.xml`. |
| **Crawled - currently not indexed** | Робот прочитав сторінку, але вирішив поки не індексувати. | На сторінці товару мало тексту, або це дубль іншої фігурки. |
| **Discovered - currently not indexed** | Робот знайшов посилання, але ще не заходив на сторінку. | Завантажено багато нових товарів, сканування відкладено. |
| **Excluded by noindex tag** | Сторінку приховано від пошуку спеціальним тегом у коді. | Це сторінка "Кошик" або "Особистий кабінет". |
| **Blocked by robots.txt** | Файл `robots.txt` прямо забороняє боту доступ сюди. | Заблоковано адмін-панель або скрипти фільтрів. |
| **Redirect error** | Помилка перенаправлення (ланцюжок редиректів або петля). | Змінили URL категорії, але налаштували редирект неправильно. |
| **404 Not Found** | Сервер відповів, що сторінки не існує. | Повністю видалили сторінку. |
| **Soft 404** | Сторінка працює (код 200), але виглядає як порожня. | Сервер замість 404 видає порожній шаблон. |

## 4. Таблиця алгоритмів Google з реальними кейсами

| Алгоритм | Рік | Що перевіряє | Кого покарали (приклад) | Що нам треба робити |
| :--- | :--- | :--- | :--- | :--- |
| **Panda** | 2011 | Якість текстів. Шукає скопійовані, занадто короткі або марні сторінки. | eBay (у 2014 році втратили 33% відвідувачів через купу сторінок без тексту). | Писати власні, унікальні описи для кожної фігурки. |
| **Penguin** | 2012 | Посилання на сайт. Шукає штучні та куплені лінки з поганих сайтів. | Expedia (сервіс втратив 25% видимості через куплені SEO-посилання). | Не купувати дешеві посилання, шукати природні згадки. |
| **BERT** | 2019 | Розуміння мови. Алгоритм почав розуміти контекст і зміст тексту, як жива людина. | Сайти, що просто спамили ключовими словами, впали в пошуку. | Писати тексти простою людською мовою, без "кривих" фраз. |

### Який алгоритм найважливіший для нас?
Для магазину **Pop Collectors** найголовніший алгоритм — **Panda**. 
В інтернет-магазинах часто бувають сторінки, де є тільки фото фігурки і ціна. Google вважає це "порожньою" сторінкою і не показує її. Тому нам треба обов'язково писати хоча б невеликі, але власні описи до кожного товару.

### Чим відрізняється BERT від Panda?
* **Panda** перевіряла лише базу: чи текст не вкрадений і чи він взагалі є.
* **BERT** почав розуміти *сенс*. Завдяки йому більше не треба неприродно вставляти фрази на кшталт *"купити фанко поп Київ недорого"*. Достатньо просто написати зрозумілий текст для покупця, і Google сам зрозуміє, про що мова.

## 7. Заповнений E-E-A-T  чек-ліст

**Experience (Досвід)**
- [x] Статті написані від першої особи або містять особистий досвід. *(Матеріали в блозі пишуться від імені реальних колекціонерів з описом їхніх вражень від фігурок).*
- [x] Є конкретні приклади, скріншоти, кейси. *(У статтях використовуються наочні приклади, наприклад, фотографії різних видів стікерів на коробках).*

**Expertise (Експертиза)**
- [x] Профіль автора підтверджує компетентність у темі. *(На сайті є інформація про авторів, яка підтверджує їхній досвід у колекціонуванні Funko Pop).*
- [x] Статті містять технічно точну інформацію. *(У блозі публікуються вузькоспеціалізовані матеріали, наприклад, детальний розбір значень стікерів: "funko-stickers-explained").*
- [x] Є посилання на авторитетні джерела. *(За необхідності використовуються відсилання до офіційних баз даних на кшталт Funko App або HobbyDB).*

**Authoritativeness (Авторитетність)**
- [x] Сторінка `/about` з описом редакції. *(Створено детальну сторінку "Про нас" із місією магазину та гарантією 100% оригінальності).*
- [x] Автори мають публічні профілі (LinkedIn/GitHub/Instagram). *(Наявні посилання на соцмережі, що підтверджують реальність авторів).*
- [x] Наявні зовнішні посилання на сайт. *(Сайт пов'язаний з офіційними соцмережами магазину).*

**Trustworthiness (Надійність)**
- [x] Сайт працює через HTTPS. *(Налаштовано безпечне з'єднання, що критично важливо для довіри покупців).*
- [x] Є контактна інформація. *(На сайті вказані способи зв'язку).*
- [x] Дати публікацій відображаються коректно. *(У статтях блогу виводиться дата, що показує актуальність інформації для ринку колекціонерів).*
- [x] Немає битих посилань. *(Навігація між статтями та головними сторінками працює коректно).*
